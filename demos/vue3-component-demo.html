分析说明

1. Vue 组件的构成

每个 .vue 单文件组件（SFC）通常包含三部分：

  • 模板（template） — 定义 HTML 结构，使用 Vue 指令和插值表达式
  • 脚本（script） — 定义组件逻辑：data、props、emits、computed、methods 等
  • 样式（style） — 可使用 scoped 属性将 CSS 限定在当前组件

本示例使用 app.component() 注册全局组件的方式，等效于 SFC 中的组件定义。

2. 组件间通信方式

  • 父 → 子（Props）：父组件通过 props 向子组件传递数据（如 :name、:color）。子组件通过 props 选项声明接收。
  • 子 → 父（Emits）：子组件通过 $emit('事件名', 参数) 触发自定义事件，父组件用 @事件名="处理函数" 监听。
  • 单向数据流：数据从父组件流向子组件（props 只读），子组件不能直接修改 props，而是通过事件通知父组件修改。

3. 本例工作流程

每个 ButtonCounter 子组件维护自身点击计数（count），点击时同时 $emit('increment') 通知父组件。父组件 App 监听 @increment 事件，累加 totalClicks。
